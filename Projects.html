Projects

By Sagar Gandhi

Below is list of projects I did.

Graphics, 3D Programming, GPU Drivers, OpenGL, OpenGL ES, Sharders, GLSL, GLSL ES

  • Debugged Intel On-Chip Graphics Driver for Windows XP using WinDbg debugger. Primarily worked on 3D HAL module. I learned most of my debugging skills on this project.
  • Programmed Polygon Reduction Tool, which reduced number of polygons by 50%-75% while maintaining original shape of the object intact. This was a whole new advancement for Game Programmers
  • Developed 3D Game on Android platform using NDK, C++, STL to help children remember the number tables. I learned how to put different modules together. At the same time, how to maintain overall performance to optimum.
  • Engineered and programmed OpenGL-ES 3D-Engine for Android which supports Advanced Obj, MD2 file formats and Audio Support. Wrote Native code using C, C++.
  • Wrote GPU Profiler for AMD using C++ on Windows 7 to compute GPU statistics at runtime. Learning about GPU internals helped visualize how to optimize the programs for visibility against resource usability.
  • Built 3D Action Game for PC from initial design to deployment, covering most of the topics in 3D Rendering. Used Qt for Windowing System Management.

Real-time Image processing, Intel Creative Camera,Intel RealSense Camera, Kinect For Windows V1, V2, OpenCV

  • Designed and Developed Domain Oriented Gesture Library for Kinect for Windows V2, which is usable in higher level languages like C# and Python.
  • Worked on Augmented Reality project for an innovative product. It targets the Education Industry directing a new way of learning.
  • Built an app for Kinect for Windows which creates an ease for presentations using MS Powerpoint. For e.g. One can traverse through slides using simple gestures. Wrote COM Office Automation program in C++.
  • Built small Game using Intel Perceptual Computing SDK, later ported it to Intel RealSense SDK
  • Created an App for Speech Impaired which would help them convey a message. Used OpenCV, C on Windows 7 platform. Porting it to Android is in progress.
  • Built 3D Action Game for PC from initial design to deployment, covering most of the topics in 3D Rendering. Used Qt for Windowing System Management.
